# Issue-Tracker
We are developing a web platform in which users can submit queries and have them answered by authorized administrators via a login portal. 

## Installation guide 
* Copy the GitHub repository in your local host.
* Install the necessary modules in your local
  * Install npm
  * Install MongoDB
  * Install other node modules.
